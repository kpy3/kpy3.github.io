Интересные ресурсы о машинном обучении

Среда, 1 января 2014

Здесь я решил собрать найденные в процессе обучения на курсе по машинному обучению ресурсы, которые показались мне интересными и могут быть полезными для углубленного изучения данной темы.

Видео

Начну с видео. Во-первых, лекции автора курса Andrew Ng по курсу Машинного обучения (CS 229) можно посмотреть на youtube (он же в режиме нонстоп).

Во-вторых, не менее интересным, на мой взгляд, является курс CS-156 профессора Yaser Abu-Mostafa из Калтеха (он же в режиме нонстоп).

Кроме обозначеных курсов доступен курс лекций по Машинному обучению, который читает Игорь Кураленок из Яндекса. Курс доступен на русском языке.

Уверен что доступных видеоматериалов гораздо больше.

Литература

Данная дисциплина набирает популярность за академическими стенами, поэтому недостатка в доступной литературе не ощущается. Тот же амазон предлагает сотни позиций на простой поиск по теме. Чтобы не запутаться и не потеряться в теме, можно следовать, например, этой рекомендации, и начать с книг попроще. Перед началом курса я полистал Machine Learning for Hackers, сейчас изучаю книгу Кефина Мёрфи Machine Learning: A Probabilistic Perspective, это обширная монография, неплохое введение и справочник по различным аспектам машинного обучения.

Сетевые ресурсы

Сетевых ресурсов по этой теме столько, что остаётся отправить заинтересованного читателя в google чтобы найти там что-то на свой вкус.
